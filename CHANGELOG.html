0.1.4 — 2026-03-17

  • Fixed: Container height not shrinking — min-height: 0 is now set inline so Elementor's own min-height doesn't block height reduction.
  • Added: Scrolled Logo Width slider (px/%) for the alternate logo image.
  • Added: Scrolled Logo Fit selector (Contain / Cover / Fill / Scale Down / None) for object-fit on the alternate logo.

0.1.3 — 2026-03-17

  • Changed: Replaced min-height + padding controls with explicit Starting Height and Sticky Height sliders for reliable height transitions.
  • Changed: New Logo CSS Class control — target any widget, image or container to shrink on scroll.
  • Changed: Removed padding shrink controls; height now drives the shrink effect directly.
  • Fixed: Container height not shrinking on scroll.
  • Fixed: Logo scale animation snapping back instead of animating when scrolling up — transition is now declared on the base state.

0.1.2 — 2026-03-17

  • Fix: z-index not applied — CSS variable was overridden by inline styles set during fixed positioning; z-index is now set inline in JS from settings value.

0.1.1 — 2026-03-16

  • Fix: Container invisible in Elementor header location — replaced position:sticky with position:fixed applied via inline JS, mirroring Elementor PRO sticky approach.
  • Fix: JS handler never initialised — replaced elementorFrontend hook (never fired) with direct DOM scanning on DOMContentLoaded + window.load.
  • Fix: Added spacer div to maintain document flow when container uses fixed positioning.
  • Fix: Added ResizeObserver to recalculate fixed width on viewport resize.

0.1.0 — 2026-03-16

  • Added: Initial plugin scaffold.
  • Added: Sticky Header extension for Elementor Container element.
  • Added: Controls — scroll threshold, transition duration, min-height, padding, logo scale, scrolled logo image, logo animation type (fade/shrink/fade-shrink), background color, box shadow, z-index.
  • Added: IntersectionObserver scroll detection (vanilla JS, no jQuery).
  • Added: CSS custom property architecture for performant, smooth transitions.
